# GymHub

## Objetivo

O GymHub é uma plataforma web desenvolvida para auxiliar usuários na organização e acompanhamento de seus treinos físicos. O sistema oferece uma interface intuitiva para facilitar o acesso a informações relacionadas à rotina de exercícios, promovendo uma melhor gestão do desempenho e da evolução dos treinos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages (Hospedagem)

## Estrutura das Páginas/Seções

### Página Inicial (Home)

* Apresentação do sistema
* Destaque das principais funcionalidades
* Navegação para as demais seções

### Área de Treinos

* Visualização dos treinos cadastrados
* Organização dos exercícios por categoria

### Área de Exercícios

* Consulta de exercícios disponíveis
* Informações detalhadas sobre execução

### Área do Usuário

* Acompanhamento de desempenho
* Gerenciamento de informações pessoais

### Contato/Suporte

* Canal para dúvidas e sugestões dos usuários

## Funcionalidades Implementadas

* Interface responsiva para diferentes dispositivos
* Navegação dinâmica entre páginas
* Organização de treinos e exercícios
* Exibição de informações de desempenho
* Design moderno voltado para praticantes de atividades físicas
* Hospedagem online através do GitHub Pages
